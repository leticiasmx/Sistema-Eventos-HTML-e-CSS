# 🎉 Sistema de Eventos

Um projeto web desenvolvido com **HTML e CSS**, com foco na organização de layout utilizando **Flexbox**.

## 📌 Sobre o projeto

O **Sistema de Eventos** é uma página web que exibe eventos de tecnologia, permitindo visualizar informações como data, local e descrição.

O objetivo principal foi aplicar conceitos de:
- Estruturação com HTML
- Estilização com CSS
- Layout moderno com **Flexbox**
- Responsividade

## 🖥️ Layout

O projeto possui:

- ✅ Header com menu de navegação  
- ✅ Banner de destaque  
- ✅ Filtros laterais  
- ✅ Lista de eventos em cards  
- ✅ Footer simples  


## 🚀 Tecnologias utilizadas

- HTML5  
- CSS3  
- Flexbox  

## ▶️ Como executar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
